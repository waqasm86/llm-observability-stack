# Lenovo/OEM Edge AI Angle

This project can support a Lenovo/OEM sponsorship conversation by demonstrating how NVIDIA-powered
Linux laptops and workstations can be shipped, evaluated, or supported as private AI edge nodes
with built-in observability.

This is a future partner validation path. It does not claim current Lenovo sponsorship,
certification, endorsement, or a commercial relationship.

## Realistic Sponsorship Ladder

1. Hardware loan or device sponsorship.
2. Joint benchmark blog.
3. Validated Lenovo NVIDIA laptop/workstation benchmark matrix.
4. Enterprise design-partner pilot.
5. OEM/SI reference architecture.

## First Ask

Provide two to three NVIDIA-powered Lenovo devices for a controlled validation program:

- RTX laptop
- mobile workstation
- desktop workstation

The project would publish a reproducible, clearly scoped benchmark and deployment report, subject
to partner review and model licensing constraints.

## Metrics to Collect

- model load time
- TTFT p50 and p95
- generated tokens per second
- VRAM usage
- GPU utilization
- power and temperature
- reliability under repeated prompts
- install time
- support and debugging workflow time

## Partner Value

- Evidence for private/offline AI use cases on Linux.
- A repeatable Kubernetes deployment and support workflow.
- Comparable performance and thermal evidence across device tiers.
- A path from individual edge devices to managed workstation and cloud GPU fleets.
- An open reference implementation that can inform an OEM/SI-supported bundle.
